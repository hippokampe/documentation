The Console !

CLI (command line interface)

All the things you do in a web-browser, you'll be able to do it right here in the console.

The available options in this current version are:

> Check your code.
> Search for a project.
> Login authentication.

PROJECTS (cli integrated navigation)

Manage your projects in the easy way.

The available and future options in this app are:

> Search for a project.
> Get a project.
> Get all the projects.
> Get the current projects.
> Read the documentation.

About

What is hippokampe?


Hippokampe it's an unnoficial cli for Holberton School. This app will allow to the students the ability to have all the features in their console.



Why hippokampe?


Hippokampe it's a complete cli app for internal purposes. It's was designed for be a efficient, easy and customizable app.



What is the focus of Hippokampe?


The focus is to bring a tool that will contribute in the productivy of the Holberton Students. Because the idea is create an app that coul be use in the console and you can do all the things that you require in order to do the projects and learn.



It's hippokampe complete?


The last version it's an alpha (v0.7.5-alpha) and it only have 4 features: check your code, get a single or multiple projects and search a project.


I will add more features but first I will extend the support to MacOs and other linux versions.



What features are related directly with the productivy?


Supported

  • Check your code
  • Generate the documentation (It will generate the documentation, but the cmd for view, it's not available)

Not supported yet

  • Read the documentation of each task
  • Read the documentation of each project
  • Generate automaticly the README.md and the prototypes
  • Search a task without the project ID
